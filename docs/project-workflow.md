# Project Workflow (Docs belépési pont)

Ez a fájl a workflow dokumentáció docs-alatti elérési pontja.

## Hol van a teljes workflow leírás?

- Elsődleges történeti/operatív leírás: `DOCUMENTATION_PROJECT_WORKFLOW.md` (repository gyökér)

## Miért van ez a fájl?

- A dokumentációs belépési pontot a `docs` mappában tartjuk.
- A jövőbeni frissítésekhez javasolt ezt a fájlt használni mint indexet, és innen hivatkozni a részletes leírásokra.
